# Dice Roller App 🎲

A simple Android Dice Roller application built using Kotlin and Jetpack Compose.

## Features

* Roll a random dice number
* Dynamic dice image updates
* Built using Jetpack Compose
* State management using `remember` and `mutableStateOf`

## Tech Stack

* Kotlin
* Jetpack Compose
* Android Studio

## Concepts Practiced

* Composable functions
* State management
* Button click handling
* Image composables
* Layouts using Column and Spacer
