# 🚀 Android Development Journey

Welcome to my Android Development Journey repository!

This repository documents my progress as I learn Android app development using **Kotlin** and **Jetpack Compose**. It contains a collection of small projects, practice applications, and hands-on exercises that help me understand Android development concepts through implementation.

## 🎯 Goal

The purpose of this repository is to:

* Learn modern Android development practices.
* Build Android applications using Jetpack Compose.
* Strengthen Kotlin programming skills.
* Understand Android architecture and best practices.
* Track my growth as an Android Developer.

## 🛠️ Technologies Used

* Kotlin

* Android Studio
* Material Design 3
